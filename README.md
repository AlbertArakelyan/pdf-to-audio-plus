## ğŸ“¦ Installation
_Note: Make sure you are using correct `node` and `yarn` versions._

### Versions
- node: `20.18.0`
- npm: `10.2.3`

### Repo
- Clone the repo
```bash
git clone https://github.com/AlbertArakelyan/pdf-to-audio-plus.git
```

### Development
- Install dependencies
```bash
npm install
```
- Run development environment
```bash
npm run dev
```
